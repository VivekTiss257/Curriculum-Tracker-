📘 Multivariate Predictive Analysis Test Atlas
Full definitions, derivations, inference logic, and examples. These tests are interrelated through the OLS framework and its assumptions.

1) t-test for Individual Coefficients

Definition: Tests if a single regression coefficient βj differs from 0 given other predictors.

t = (β̂j - 0) / SE(β̂j)
Derivation (Core)
β̂ = (XᵀX)⁻¹Xᵀy
Var(β̂j) = σ² (XᵀX)⁻¹ⱼⱼ
SE(β̂j) = √Var(β̂j)
⇒ t = (β̂j − 0) / SE(β̂j)
Compare with t_(n-k-1)
Inference: |t| > t_critical ⇒ reject H0 (βj = 0). Use p-value to quantify evidence.

2) F-test (Overall Significance)

Definition: Tests whether the full model explains more variance than an intercept-only model.

F = (SSR/k) / (SSE/(n-k-1))
Derivation
Total: SST = SSR + SSE
Mean Squares: MSR = SSR/k
MSE = SSE/(n-k-1)
F = MSR / MSE
Inference: Large F ⇒ at least one predictor matters.

3) VIF (Multicollinearity Test)

Definition: Measures how much variance of β̂j is inflated due to multicollinearity.

VIFj = 1 / (1 - Rj²)
Derivation
Var(β̂j) ∝ 1/(1-Rj²)
So inflation factor = 1/(1-Rj²)
Inference: VIF > 5 (moderate), >10 (serious).

4) Breusch–Pagan Test

Definition: Detects heteroscedasticity (variance of errors depends on X).

LM = n·R² (from aux regression of residual² on X)
Derivation
1) Fit regression, get residuals ê
2) Regress ê² on predictors X
3) LM = n·R² ~ χ²(k)
Inference: If LM > χ²_crit ⇒ heteroscedasticity exists.

5) White Test

Definition: General heteroscedasticity test including nonlinear terms & interactions.

LM = n·R² from regression of ê² on X, X², XᵢXⱼ 
1) Fit model, get ê
2) Regress ê² on X, X², cross terms
3) LM = n·R² ~ χ²(df)
Inference: Reject H0 ⇒ variance is not constant.

6) Durbin–Watson

Definition: Tests autocorrelation in residuals.

DW = Σ(êₜ - êₜ₋₁)² / Σêₜ² 
DW ≈ 2(1 - ρ)
DW ≈ 2 ⇒ no autocorrelation
Inference: DW < 2 ⇒ positive autocorrelation.

7) Jarque–Bera

Definition: Tests if residuals are normally distributed.

JB = (n/6)(S² + (K-3)²/4) 
S = skewness, K = kurtosis
JB ~ χ²(2)
Inference: Large JB ⇒ non-normal residuals.

8) Ramsey RESET

Definition: Tests functional form misspecification.

Add ŷ², ŷ³ to regression, test if significant 
H0: added nonlinear terms = 0
Use F-test on augmented model
Inference: Reject H0 ⇒ missing nonlinear structure.

9) Hausman Test

Definition: Compares Fixed Effects vs Random Effects in panel data.

H = (β_FE - β_RE)ᵀ [Var(β_FE) - Var(β_RE)]⁻¹ (β_FE - β_RE) 
H ~ χ²(k)
If significant ⇒ FE is consistent, RE is biased
Inference: Reject H0 ⇒ choose FE.

10) ADF (Stationarity)

Definition: Tests unit root in time series regression.

Δyₜ = α + βt + γyₜ₋₁ + ΣδᵢΔyₜ₋ᵢ + εₜ
Inference: H0: γ = 0 (unit root). Reject ⇒ stationary.
📊 Example: Residual Variance vs Fitted
These tests are connected: OLS relies on assumptions; each test probes one assumption. Violations (heteroscedasticity, autocorrelation, misspecification) distort t/F inference.
