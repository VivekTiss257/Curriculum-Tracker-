🧪 Hypothesis Testing ✓ Done

The scientific backbone: null/alternate hypothesis, Type I/II errors, p-values, significance levels.

H₀ / H₁p-valueα level Type I ErrorType II ErrorPower
p-value < α → Reject H₀
Type I Error (α): False Positive — rejecting a true H₀
Type II Error (β): False Negative — failing to reject false H₀
Power = 1 - β
💡 p-value is NOT the probability that H₀ is true. It's the probability of observing the data assuming H₀ is true.

🌊 Central Limit Theorem ✓ Done

The single most important theorem in statistics. Sample means approach normality as n→∞, regardless of the population distribution.

CLTSampling Distribution Standard ErrorLaw of Large Numbers

Logit, Probit, LPM — Key Differences

💡 The Paradigm Shift That Unlocks Probabilistic ML

Frequentist stats treats parameters as fixed constants. Bayesian treats them as random variables with probability distributions. This simple shift lets you: (1) encode domain knowledge as priors, (2) quantify uncertainty naturally, (3) update beliefs sequentially. It's the foundation of probabilistic ML, Bayesian neural networks, and A/B testing with proper uncertainty quantification.

Philosophical Discussion: Belief vs Frequency Bayes Theorem Prior Distributions (conjugate priors) Likelihood Function Posterior Distribution Bayesian Decision Theory Credible Intervals (vs Confidence Intervals) MCMC (Metropolis-Hastings, Gibbs) Direct Sampling & Importance Sampling PyMC / Stan Implementation Model Comparison (WAIC / LOO) Bayesian GLM / State-Space Models
The Bayesian Core
P(θ | data) ∝ P(data | θ) × P(θ) ← Posterior ∝ Likelihood × Prior Bayes CI: P(θ ∈ [a,b] | data) = 0.95 ← TRUE probability statement! vs Frequentist CI: "95% of such intervals contain the true θ" ← very different Conjugate: Beta prior + Binomial likelihood → Beta posterior
💡 Bayesian A/B testing is strictly better than frequentist for business: you get P(variant B > variant A | observed data) — a direct probability statement. You can also stop the test early without inflating false positives.
